# Font 
### Standard size commands
```
\tiny
\scriptsize
\footnotesize
\small
\normalsize (default)
\large
\Large
\LARGE
\huge
\Huge
```

## Define a new command using `\newcommand`  
```
\newcommand{\commandname}[number of arguments]{definition}
```


##### Custom command for vertical spacing
```
\newcommand{\vspacecustom}{\vspace{1em}} 
```
